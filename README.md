## 🏗️ 项目架构

### MVVM架构分层

```
┌─────────────────────────────────────────────┐
│                  UI 层                       │
│  ┌─────────┬─────────┬─────────┬─────────┐  │
│  │ 角色页面 │ 聊天页面 │ 配置页面 │ 设置页面 │  │
│  └─────────┴─────────┴─────────┴─────────┘  │
└─────────────────────────────────────────────┘
                      ↓
┌─────────────────────────────────────────────┐
│              ViewModel 层                    │
│  ┌─────────────┬─────────────┬───────────┐  │
│  │CharacterVM  │  ChatVM     │ ConfigVM  │  │
│  └─────────────┴─────────────┴───────────┘  │
└─────────────────────────────────────────────┘
                      ↓
┌─────────────────────────────────────────────┐
│             Repository 层                    │
│  ┌─────────────┬─────────────┬───────────┐  │
│  │CharacterRepo│ MessageRepo │ ApiConfig │  │
│  └─────────────┴─────────────┴───────────┘  │
└─────────────────────────────────────────────┘
                      ↓
┌─────────────────────────────────────────────┐
│              Data Source 层                  │
│  ┌─────────────────┬─────────────────────┐  │
│  │   Room Database │    Retrofit Client   │  │
│  │   (本地数据)     │      (网络请求)       │  │
│  └─────────────────┴─────────────────────┘  │
└─────────────────────────────────────────────┘
```

## 🚀 快速开始

### 环境要求

- Android Studio Hedgehog 或更高版本
- JDK 17 或更高版本
- Android SDK 34
- Gradle 8.2 或更高版本
- Android 手机或模拟器（API 24+）

### 运行步骤

1. **克隆项目**
   ```bash
   git clone <仓库地址>
   cd chatbot-android
   ```

2. **用Android Studio打开**
   - File → Open → 选择项目文件夹
   - 等待Gradle同步完成

3. **配置模拟器或真机**
   - 推荐使用真机调试（更流畅）
   - 真机需开启USB调试

4. **运行应用**
   - 点击绿色运行按钮 ▶️
   - 选择目标设备
   - 等待编译安装
