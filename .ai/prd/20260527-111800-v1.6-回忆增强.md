# PRD: v1.6「回忆增强」

> Generated: 2026-05-27 11:18:00

## 1. 目标用户/场景

**目标用户**: 想要搜索聊天历史、收藏重要消息的用户
**使用场景**:
- 用户想要找到之前和某个角色讨论过的某个话题
- 用户想要收藏某条重要的回复
- 用户想要导出某段聊天记录

## 2. 功能列表

### MVP 功能

#### 聊天全文搜索
- [ ] 在当前角色聊天页添加搜索框
- [ ] 支持搜索消息内容
- [ ] 搜索结果高亮显示
- [ ] 点击结果跳转到对应消息

#### 按时间筛选
- [ ] 添加时间范围筛选器
- [ ] 预设选项：今天、本周、本月、更早
- [ ] 支持自定义日期范围

#### 消息收藏/标记
- [ ] 长按消息添加「收藏」选项
- [ ] 收藏的消息在单独页面展示
- [ ] 支持取消收藏

#### 聊天导出
- [ ] 导出当前角色聊天为 Markdown
- [ ] 导出当前角色聊天为 TXT
- [ ] 支持导出到文件或分享

1. 用户在聊天页点击搜索图标
2. 输入关键词
3. 查看搜索结果
4. 点击结果跳转到对应位置
