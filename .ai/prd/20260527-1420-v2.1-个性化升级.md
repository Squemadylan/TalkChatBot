> Generated: 2026-05-27 14:20:00

## 1. 目标用户/场景

**目标用户**：追求视觉体验、重视个性化表达的深度用户
**使用场景**：
- 用户想自定义聊天界面外观（背景、气泡颜色、字体）
- 用户偏好特定阅读体验（字体大小、行间距）
- 用户喜欢 AI 回复有打字机效果增加沉浸感
- 用户遇到外语回复想要即时翻译

**用户痛点**：
- 界面千篇一律，没有个性化选项
- 气泡颜色固定，无法搭配背景
- 默认字体大小不适配小屏/大屏用户
- 外语 AI 回复阅读困难

## 3. 数据模型

| 实体 | 字段 | 说明 |
|------|------|------|
| SharedPreferences | chat_background_type | 0=内置, 1=图片, 2=纯色 |
| SharedPreferences | chat_background_value | 颜色值或图片路径 |
| SharedPreferences | user_bubble_color | 用户气泡颜色 hex |
| SharedPreferences | assistant_bubble_color | 助手气泡颜色 hex |
| SharedPreferences | font_size | 字体大小 sp 值 |
| SharedPreferences | line_spacing | 0=紧凑, 1=标准, 2=宽松 |
| SharedPreferences | dark_mode_style | 0=普通深色, 1=OLED黑 |
| SharedPreferences | typewriter_enabled | 0=关闭, 1=开启 |
| SharedPreferences | typewriter_speed | 每字毫秒数 |
| SharedPreferences | translate_enabled | 0=关闭, 1=开启 |
| SharedPreferences | translate_language | 目标语言代码 |
| Character | background_custom | 角色自定义背景 |
| Character | fontSize | 角色字体大小 |
| Character | typewriterEnabled | 角色打字机开关 |

## 5. 用户旅程

### 旅程1: 自定义气泡颜色
1. 用户进入「设置」→「气泡颜色」
2. 选择配色方案或点击「自定义」
3. 颜色选择器选择用户气泡和助手气泡颜色
4. 实时预览效果
5. 保存，返回聊天页生效

### 旅程2: 开启打字机动画
1. 用户进入角色对话
2. AI 开始回复，显示逐字打印效果
3. 用户点击跳过，完整内容立即显示
4. 用户不喜欢可在设置关闭

### 旅程3: 翻译消息
1. AI 回复完成后，消息下方出现「译」按钮
2. 用户点击「译」
3. 翻译结果以浅色小字展示在消息下方
4. 用户再次点击收起翻译

## 6. 非功能需求

- **性能**：UI 样式切换 < 100ms，动画流畅 60fps
- **兼容性**：Android 8.0+，不同屏幕密度下效果一致
- **存储**：仅使用 SharedPreferences，无新数据库表
- **功耗**：打字机动画对电池影响极小

## 7. 风险与边界

> ⚠️ Assumption：翻译依赖第三方 API，免费版可能有频率限制
> ⚠️ Assumption：自定义背景图片需要处理内存占用，建议压缩
> ⚠️ Assumption：部分 OEM 设备深色模式实现不一致，OLED 模式需要额外适配

**风险**：
- 翻译 API 有调用限制，需考虑降级方案
- 字体大小过大可能导致布局溢出
- 长时间打字机动画耗电
